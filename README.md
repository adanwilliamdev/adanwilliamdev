## 🚀 Sobre Mim

Sou um desenvolvedor Full Stack apaixonado por tecnologia, infraestrutura e desenvolvimento de software.

Tenho experiência na construção de aplicações web modernas, APIs robustas, sistemas de monitoramento e dashboards para ambientes de TI. Gosto de transformar problemas complexos em soluções simples, eficientes e escaláveis.

Atualmente estou aprofundando meus conhecimentos em cloud computing, microsserviços, observabilidade e automação de infraestrutura.

### Atualmente

* 🔭 Desenvolvendo o **OrbNOC**, plataforma de monitoramento e observabilidade
* 🎬 Desenvolvendo o **Trackflix**, sistema de gerenciamento de filmes e séries
* 🌱 Estudando Kubernetes, AWS e Arquitetura de Microsserviços
* 🤝 Aberto para colaborações em projetos Open Source
* ⚙️ Explorando DevOps, Containers e Cloud

## 📌 Projetos em Destaque

### 🚀 OrbNOC

Plataforma de monitoramento e observabilidade voltada para ambientes NOC e infraestrutura de TI.

**Principais funcionalidades:**

* Monitoramento de hosts
* Verificação de disponibilidade
* Métricas em tempo real
* Dashboard responsivo
* Histórico de incidentes
* Exportação de relatórios
* API REST
* Sistema de alertas

### 🎬 Trackflix

Aplicação para gerenciamento e acompanhamento de filmes e séries.

**Principais funcionalidades:**

* Catálogo personalizado
* Controle de progresso
* Busca avançada
* Interface moderna
* Histórico de visualização
